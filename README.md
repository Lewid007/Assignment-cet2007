# Cet2007 Assigment Iot Device Monitor system

This is the git hub repostitary for my assignment for module cet2007 Systems design and development.
## Overview of the project
This project will simulate a network monitoring system and allow you to add remove, edit, search and sort devices within the system.

## How to run
When running this project you need all files to be downloaded into the same folder for the solution to run due to different aspects of the solution (classes) are spread across multiple files that work together to form one large solution.
